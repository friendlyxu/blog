1. 列表接口

请求参数

参数名 类型 必填 描述 默认值
u string 是 用户登录参数 null
token string 是 用户登录参数 null
channel int 是 消息频道，1互动消息，2点赞消息 1
page int 否 页码 1

返回示例

返回空数据示例

返回参数

参数名 类型 必填 描述
data.list.comment_id int 否 评论id
data.list.reply_id int 否 回复id
data.list.to_reply_id int 否 被回复id
data.list.game_id int 是 游戏id
data.list.from_info string 否 评论/回复的内容，type=2点赞类型，没有该字段
data.list.from_status int 否 作品删除状态，0删除，1未删，与from_info字段同时出现
data.list.to_info string 否 被评论/回复的内容, type=1评论作品，没有该字段
data.list.to_status int 否 作品删除状态，0删除，1未删，与to_info字段同时出现
data.list.game_info string 是 作品名称
data.list.game_status int 是 作品删除状态，0删除，1未删，与game_info字段同时出现
data.list.time int 是 完整的发布时间格式
