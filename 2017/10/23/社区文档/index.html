    社区消息

    ##背景描述

    • 站内信的前身

    • redis部分采用有序集存储用户列表，哈希存储消息主体，

    • 每天检查清理三个月前的数据

    • 未读消息ID存储有序集，已读消息ID则从未读有序集合删除

    • mysql部分采用季度分表实现，根据脚本每季度最后一天自动建表

    device_name string 是 设备名称 Google Nexus S
    device_system_version string 是 系统版本号 4.1.1
    mode int 是 (1话题回复消息，2小编提醒消息，3AT提醒消息), 1
    page int 否 分页 1
    page_size int 否 每页显示数量 10
