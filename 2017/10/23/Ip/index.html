##PC 121.9.235.205/119.38.129.212

  • 查看系统负载 top
  • 实时查看apache的php报错 tail -f /tmp/php54.err
  • 实时查看crontab的php报错 tail -f /tmp/php.err
  • 查看apache访问日志 /var/logs/apache2410
  • 查看系统日志，系统日志为：/var/log/messages
  • pc更新代码到线上正式环境 改到9服务器更新PC环境代码
    1.vim /cron/filepath 编辑这个文件，把要代码更新的文件路径写入到这个文件
    2.执行sh /cron/tbsomefiles.sh 进行文件更新。
  • pc还原代码（当更新代码到线上正式环境出问题时）
    1.执行cd /usr/local/backup/webbak，来到代码备份目录。
    2.使用tar zxvf 1zhe.com_xxxxxx.tar.gz，解压一份最新的备份。
    3.vim /cron/berestoredfilepath，编辑这个文件，把要还原代码的文件路径写入到这个文件
    4.执行 sh /cron/restoresomefiles.sh 进行文件恢复。
  • 连接memcache Telnet 192.168.1.205 11211
